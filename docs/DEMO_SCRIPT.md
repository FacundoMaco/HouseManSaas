# Demo Script - HouseMen SaaS
## For Woodland Real Estate

---

## Demo Structure (20-30 minutes)

### 1) Introduction (2 minutes)

- Brief context: focus on laundry operations and coordination pain
- Set objective: show how a simple dashboard improves visibility and flow

---

### 2) The Problem (3 minutes)

- Coordination by radio or walk-around checks
- No real-time view of machine status
- Small delays compound across the day

---

### 3) Live Demo (12-15 minutes)

Focus on three core flows:

**A) Dashboard overview**
- Active loads and current statuses
- Clear timers and next actions
- Optional: daily checklist if time allows

**B) Create a load and start washer timer**
- Select load type, add note (optional)
- Start washer timer
- Point out the visible timer and status

**C) Move to dryer and complete the load**
- Start dryer, auto-assign available unit
- Show dryer timer and status
- Mark load complete

---

### 4) Benefits and ROI (3 minutes)

- Emphasize measurable outcomes
- Note that ROI is calculated from pilot data
- Highlight improved visibility and fewer coordination steps

---

### 5) Future Roadmap - IoT (2 minutes)

- Overview of sensor-based automation as a next step
- Position as optional after pilot validation

---

### 6) Implementation (2 minutes)

- 1-2 day setup and training
- Go-live immediately after training

---

### 7) Q and A + Close (5 minutes)

- Confirm interest in pilot
- Set next meeting to scope pilot and success metrics

---

Estimated duration: 20-30 minutes  
Prepared for: Woodland Real Estate
